Algorithm notes

This page uses the v18 JavaScript BigInt recognizer as the primary engine, with Pyodide kept as a fallback/cross-check. The JS engine implements exact rational arithmetic, coefficient-list parsing, long Weierstrass recognition, rational-point plane-cubic transformation, reduced Q-minimal model recovery, and Aronhold ternary-cubic j-invariant fallback. If Pyodide is available, the Python v17 core may upgrade or verify the JS result.

Important: a same-j atlas hit is only a same-j candidate. The page labels a result as an exact model match only when the recognizer produces a minimal coefficient vector matching a stored curve.
