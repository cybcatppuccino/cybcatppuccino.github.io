About this build

This version uses a fast metadata layer for search, lazy-loads vertex chunks only when a record is opened, and keeps calculations close to the vertex data. Some global AGL2(ℤ) canonicalization and all-record merging remain conservative by design; drawn searches and adjacency navigation perform stricter equivalence checks on narrowed candidate sets.
