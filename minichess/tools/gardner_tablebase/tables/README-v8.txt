Gardner MiniChess v8 tablebase mount point
==========================================

Keep the files generated by the v7/v7.1 Python builder in this directory.
The v8 patch intentionally does not include or replace the generated data.

Required for the exact core:
  manifest.json
  <signature>/metadata.json
  <signature>/*.wdl.gz
  <signature>/*.dtm.gz

Optional practical overlay:
  practical-manifest.json
  practical/<signature>/metadata.json
  practical/<signature>/*.idx.gz
  practical/<signature>/*.val.gz

The browser loads only the material class and gzip block required by the
current <=6-piece position. Unknown or absent records fall back to the JS
Alpha-Beta engine.
