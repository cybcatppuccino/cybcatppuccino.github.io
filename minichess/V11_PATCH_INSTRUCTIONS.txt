Gardner MiniChess v11 patch

Apply this patch over a complete v10.2 installation only.

Included files are changed/new files only. Existing PGN archives, PDF references,
local tablebase blocks and unchanged assets are intentionally excluded.

After copying:
1. Restart the local web server.
2. Force-refresh the browser.
3. If testing tablebases, keep your existing tools/gardner_tablebase/tables/ directory.

The persistent browser analysis cache is intentionally reset to gardner-analysis-cache-v11.
