Gardner MiniChess v10.1 incremental patch

1. Start from a complete v10 installation.
2. Copy every file in this patch over the project root, preserving folders.
3. Do NOT delete or replace tools/gardner_tablebase/tables/.
4. The patch intentionally contains no PGN, PDF, or generated tablebase gzip files.
5. Restart the local HTTP server and force-refresh the browser once. v10.1 uses a new analysis-cache key, so stale v10 mate/draw evaluations will not be reused.

Run:
  python -m http.server 8000

Open:
  http://localhost:8000

Recommended regression checks:
  node tests/v10_1-mate-tests.mjs
  node tests/v10-low-progress-draw-tests.mjs
  node tests/play-worker-tests.mjs
