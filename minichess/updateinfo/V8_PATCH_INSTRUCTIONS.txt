Gardner MiniChess v8 patch
==========================

This archive contains only files changed or added since v7.1.

Installation
------------
1. Stop the local web server.
2. Back up the v7.1 project.
3. Copy the CONTENTS of this patch folder over the v7.1 project root.
4. Keep the generated database already present under:

   tools/gardner_tablebase/tables/

   Do not delete manifest.json, practical-manifest.json, material folders,
   or any .gz blocks. The patch adds only README-v8.txt there.
5. Restart the local server and hard-refresh the browser page.

The v8 browser reader expects the custom GardnerTB/GardnerPracticalTB format
created by the supplied Python generator. It does not read orthodox Syzygy
.rtbw/.rtbz files.
