# cybcat's homepage

Static GitHub Pages site.

## Pages
- `index.html` — redesigned homepage with random weight 2 / weight 4 newform card.
- `test.html` — HPDB introduction plus database search page.
- `hypergeometric-motives.html` — table for Hodge type `(1,1,1,1)` and paramodular forms.
- `ries.html` — browser RIES-lite v6.2 page with level up to 9, exact integer shortform search up to effort 7, richer structured integer database families, tiny pretty-residual database, decimal/complex algebraic-number recognition, high-precision expression evaluation, and bounded integer factorization.
- `puzzleday.html` — standalone playable puzzleday page.
- `pool.html` — pool simulator.
- `mine/index.html` — AI minesweeper.

`hadamard.html` is retained only as a redirect to `test.html` for old links.


## RIES v6.2

RIES v6.2 separates high-precision expression output from RIES candidates, expands the default small-degree algebraic sweep to degree 8, and adds an on-demand number-analysis panel for continued fractions, base conversions, and digit statistics.
