RIES

A richer browser-side RIES-inspired search panel. It supports expression search, equation search near a target root, symbol selection, original-style -S/-N filters, and downloadable full RIES source/manual files.

Run in the browser

This JavaScript version is still a lightweight static-page approximation of the original C program, but it now exposes more of the original workflow: choose symbols, restrict the search, and search either expressions close to x or equations whose root is close to x.
